UOA Installation


A) If you want to play on an UOA server on Internet :

1) download the haks (CEP v2.2 / CCP v2.2 / CSP v2.2 / JXPs / UOAs) and extract/install them into your Nwn/hak folder.
2) choose an UOA server on the Nwn Gamespy and connect !

 

B) If you want to host an UOA server or to play alone (local) :

b1) Basic installation :

1) download the module (UOA) and extract it into your Nwn/modules folder.
2) download the haks (CEP v2.2 / CCP v2.2 / CSP v2.2 / JXPs / UOAs) and extract/install them into your Nwn/hak folder.
3) download the 3 NWNX files (NWNX (nwnx-2.7-beta4-win32-full.rar) / NWNX (nwnx_odbc2_0924.zip) / NWNX (addicteds_TMI.zip)) and install/extract them into your Nwn folder.
4) download this file and place it in your Nwn folder. Say "Yes" when it asks you to overwrite the old file with the new one.
5) At this point, you can start to play by launching the "NWNX2.exe" file in your Nwn folder.

 

b2) To use a MySQL database :

6) The Arlandia default package basically uses a SQlite3 database within NWNX. But if you want to use a MySQL database instead :

- MySQL on Internet : connect to your MySQL manager and create a "uoa" database. Then configure NWNX accordingly to your MySQL database provider.
- MySQL on Local : download the WAMP server (WampServer) and install it into its default directory (mostly C:\Program Files\Wamp). Run it ! Then connect to the MySQL manager of WAMP ("phpMyAdmin") and create a new user "uoa" with password "uoa". Then create a "uoa" database.

7) Open the "NWNX.ini" file from your Nwn folder and add a ";" in front of following lines :

source = sqlite
file = uoa.db

then remove the ";" in front of the following lines :

;source = mysql
;server = localhost
;user = uoa
;pwd = uoa
;db = uoa

8) Go to the windows firewall manager and open the port "TCP 3306".

 

b3) To use the UOA website model :

9) To have the website to work you MUST use a MySQL database. See section 6.
10) download the website model (UOA website v1.0 (coming soon)) and extract it in your "C:\Program Files\Wamp\www" folder.
11) the default DM password is "dm". If you want to change it, open the "index.php" and "universe.php" files and change the "$dmlogin" variable.
12) To see the website, point the mouse on the WAMP icon and choose "Localhost". Enjoy !
